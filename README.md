This repo contains the entire source code for the Amica website.

Setup instructions
1. `git clone https://github.com/timstoddard/amica-website.git`
2. `cd amica-website`
3. `npm install`
4. `npm start`

Technologies used:
* ReactJS
* React Router
* Node/NPM
* Webpack
* SCSS
* ESLint
* Express (for production server)
